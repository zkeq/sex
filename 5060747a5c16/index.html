 # 以形补形？你的身体可不同意 # 



说到壮阳养生，最值得一提的便是各色各样的“动物鞭与其蛋蛋”，大到星级酒店、小到大排档，几乎都能见到它们软趴趴静卧在面前的盘子里的身影。不管是出于好奇尝试抑或确信其能滋补，不少年轻人的食谱里也默默开始加入诸如“牛鞭”、“牛蛋”等菜品。


“听说这些能壮阳嘛，趁年轻以形补形总是好的。”


“吃啥补啥，以形补形”的说法似乎已成为生活中必不可少的“常识”，平凡如骨头汤对骨折的恢复都起到了不小的效果，更何况是雄性激素和睾丸酮丰富的生殖器官，想必壮阳效果一定很不错吧？


但有实验已证明，动物生殖器在经过高温烹饪之后，其中所拥有的激素会严重失活，即便存有少部分“幸存者”，也将在“旅程”中连同其他蛋白质一起被胃酸扼杀在消化道里，分解成最基础的单位。



 # 惹了满口味儿，还是蜡枪头 # 



韭菜可谓是植物界里壮阳的代言人，其特殊的辛香味儿也让其成为了校门口烧烤摊的宠儿。常有听闻“韭菜对男性好，这可是老中医说的。”

老中医os：诶？这锅我不背


中医有言：“韭菜性温，味辛，补肾温阳，有益阳之效，而无壮阳成分。”


“益阳”乃是温养，与“壮阳”还是有着不小的差异。而又有人说韭菜含有对生长发育和生殖影响颇大的锌元素，进食韭菜能促进性能力的增强。但每百克韭菜仅含有0.43毫克的锌，想要达到肉眼可见的壮阳，是否需要如食草动物一般每日大量啃食韭菜呢？


况且，韭菜也并非多多益善，《本草纲目》曾记载：“韭菜多食则神昏目暗，酒后尤忌。”可见古人也并不主张多吃韭菜，所谓“老中医的说法”不过是民间流传下来的谬论罢了。



 # 别以为只长得像，我是真有料 # 



男生烧烤时韭菜可以不吃、酒可以不喝，但面前一定会摆放着几只生蚝。


“嘿嘿，这玩意儿吃了对男人好。”眼前的男生贼兮兮的笑着，递了只生蚝给我。又是一个听说能壮阳的食物？不过，这回还真是个好东西。


人们对“贝类能壮阳”的崇拜最早源自于其外形与肾脏相似，但如今也不仅仅于此。生蚝含有丰富的微量元素，其锌元素含量之多远非韭菜之辈能相比。


作为生长发育必不可少的元素，锌对睾丸酮水平的提升有着至关重要的作用。医学表明，锌大量存在于男性睾丸之中，并参与了精子形成到成熟的全过程。若是缺乏锌元素，青少年会出现不发育、第二性征不出现的情况，而成年人则有可能出现精子质量下降、勃起障碍等生殖障碍。


除去生蚝以外，以贝类为首的大部分海产品都能给我们提供足量的锌元素，换句话来说也就是能在一定程度上起到壮阳的效果。不过，就算微量元素多如生蚝，也要吃50只以上的才能起到肉眼可见的显著效果。且不说能否塞得下，一次性吃那么多也怕是会代谢紊乱吧。

 # 吃我吃我，不仅医生远离我 # 



自亚当与夏娃相会于伊甸园第一颗“禁果树”下，香兰与蛇缠绕，如性一般迷人与危险，这也让主流认为的“禁果”——苹果成为了诱惑的代名词。


事实也正是如此，苹果是我们最常见的催情食物之一。经研究人员研究发现，苹果中含有多种多酚物质和抗氧化剂，这些物质有利于刺激私处血液的流动，增强性兴奋。不仅如此，苹果中还含有与女性雌激素——雌二醇十分相似的植物雌激素根皮苷，这对女性性唤起起到了重要作用。


据英国《每日邮报》报道，意大利的一项研究发现，苹果不仅是一种低热量零食，同时也是女性的催情食物，可提高女性性功能，提升性快感。每天一个苹果，不仅医生远离我，性无力也远离我。


莓辣建议大家，不妨每天一个苹果，一个月后来一场干柴烈火，感受性的一把火，燃烧整个胴体吧。




 # 来吧来吧，一起纵享丝滑 # 



说到爱情，头脑风暴出的东西定是多种多样，但有一项一定不会少，那便是巧克力。


从某种意义上来说，巧克力可谓是“可食用爱情”。一项生理学家研究显示，大脑对爱情的反应与品尝巧克力后产生的反应是一样的，所以人们都说巧克力是爱情的味道。

（单身狗们赶紧买起啊，感受“爱神”的降临）


不仅如此，巧克力中的化学物质——色氨酸在人体使用过后，可让大脑感到一丝丝愉悦的感觉，有助于人体缓解疲劳和疼痛，让人处于一种轻松愉悦的状态，会让人更期待两性之乐。


所以当你发现自己减肥已久的女朋友毫无缘故的吃起了数十块巧克力，出门买点烤生蚝吧：）



 # 甜蜜蜜，你笑的多甜蜜 # 



“两只小蜜蜂啊，飞到花丛中啊”蜜蜂勤劳如你我，所酿之物不有些“奇效”，也是难以说得过去。


这不，研究人员就发现蜂蜜中含有生殖腺内分泌素，具有明显的能使性腺活跃的生物活性。而且，蜂糖极易被血液吸收，对精液的形成十分有益。


而蜂王浆中则含有天门冬氨酸这种“助性”物质，能促进发育，提高性功能，刺激生殖能力，同时增强机体抵抗力，促进新生代谢和改善造血功能。


顿时觉得，蜂蜜才是真正的“黄金液体”，别说了，今晚就冲蜂蜜当晚餐饮品吧。
