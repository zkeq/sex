我在大学图书馆看到张竞生编撰的《性史》，里面写到一些人在儿童时期玩过家家游戏模仿大人结婚送入洞房的情节。


男孩依葫芦画瓢地把荫茎放进女孩的阴户里，尽管懵懂无知的孩子并不理解其代表的意思。

 


文章的叙述笔调很平静，似乎这个现象很常见。但它一下子让我想起发生在六岁前，我以为自己早已忘记的事情。



1


上小学前的日子是在农村里度过的。我的父母一整天都在镇上的水泥厂里打工，晚上才回来，于是白天家里只有我和爷爷奶奶。爷爷身体不好，常年卧床，所以只有奶奶照看我。


奶奶是个目不识丁的乡野农妇，脾气暴躁，手脚麻利。她给我洗澡时非常用力地搓我的身体，像洗一篮衣服似的，三两下就搞定了。她还经常把洗发水误当成沐浴露抹在我身上，但其实这不算什么。


因为连我和同村的小男孩玩脱裤子的游戏，她都不知道。


via / www.cc0.cn


第一次发生在我家的新屋里，新屋和旧屋并列在一起，中间隔着一条窄过道。说是新屋，其实我们一直都住在旧屋里，新屋空荡荡的，堆了一些没什么用的杂物，落满灰尘。


那时我大概五岁，一起玩的还有两个与我年龄相仿的伙伴，一男一女，男孩叫小豹，女孩叫碗妹。


我们在新屋最靠里的房间内玩，小豹忽然脱下自己的裤子，露出一小截幼嫩白皙的阴茎，他说：“你们也脱裤子吧，我们这样玩玩。”


我和碗妹顺从地脱下裤子，她神色平常，我也没有任何羞耻的感觉。但心里有点惴惴不安，担心奶奶发现我们在做这种事，可又不太明白这种事代表着什么。


我们两个小女孩按照他的话，轮流上前将自己撒尿的地方贴到他的荫茎上，随后小豹做出下胯向前一顶一顶的动作。正面之后，再换用背面对着他。


身体好像有点舒服的感觉，穿上裤子后，我觉得挺好玩的，心里涌出一种新鲜的奇特感，三个小孩笑嘻嘻地还约了什么时候再玩。

2


这种三人的脱裤子游戏玩了好几次后，有天碗妹不在村子里，小豹把我带回他家，进入他的卧室里，把门锁上。比他大五岁的哥哥正躺在靠墙的那一铺床上。


回忆到这里，像断裂了，又像打了马赛克般一团模糊。


忘了我的裤子是怎么脱下来的，忘了自己有没有说话。但我没有叫喊，呆呆地看着小豹的哥哥跳下床，他双手如钳子一般禁锢着我的身体，我立在原地不能动。


如果有个上帝视角，画面一定是这样的：十五年前的那个夏日，密闭的房间里只有小女孩和一大一小两个男孩，男孩弯下腰拼命试图把他那更大一点的荫茎插进她下面，她的眼神迷茫惊慌。


via / Loputyn


直到他裸着下身躺回床上，叫我含住他的荫茎时，我粗心大意的奶奶终于想起那个不知道跑哪玩去的调皮孙女。


这时，她刚好来到屋外，听小豹的奶奶说我们在房间里玩，于是在外面拍打窗户，问我们在干嘛。


小豹哥哥还没来得及也不打算穿上裤子，语气平稳地隔着窗户对我奶奶说，“没什么，我们在玩上课的游戏呢，教教小妹识字。”


他的从容淡定超脱年龄，完全不像一个才十岁的孩子，边说边给我使眼色，让我冲着窗户解释几句，“我们真的在玩而已”。


小豹这时附在我的耳边小声地说：“如果你奶奶发现我们在做这些事，她会打死你的。”我已经挨过无数个她的大巴掌了，心生恐惧，于是照做了。


奶奶不仅没有走进房间里，甚至连我们没打开窗也不感到奇怪，听到我发出的声音后，她爽朗地笑了几声就走了。


而我忽然觉得这个游戏变味了，有点心慌不安以及遭人强迫的委屈，一点都不好玩了。小豹的哥哥仍在强迫我含他的阴茎，我迟迟不肯把嘴送下去。


他见状，叹了口气，握着荫茎把它挪到一侧，指着睾丸旁边的皮肤：“你亲这里好了。”像是让步的商量口吻，又带有隐隐约约的威胁，没说出来的意思好像是：


“如果你不照做，就别想走出这个房间。”


我害怕了，只好乖巧地往他大腿根部亲下去，他却抓着我的头，强迫我往荫茎亲去，一口一口地扯过去，直至碰到荫茎根部的边缘为止。


我没敢挣扎。他的体型几乎比我大一倍，害怕他会动手打我，因为他的语气已经很强硬了。


也许是感觉到我的动作发僵，他的声音从头顶上方传来：“就亲几下而已啦，没什么大不了的，我们村那个珠妹也亲过我了。”


听到珠妹的名字，我像得到莫大的宽慰。在大人的嘴里，她就是那种别人家的孩子，说什么做什么都能赚到一身夸。珠妹不怎么和我玩，虽然都是农村娃，但她文文静静的，甚至有点城市小孩的“高贵气质”。


小豹他哥的威逼态度让我感觉屈辱，但珠妹居然也这样做过！那应该不算太坏的事情吧。想到这里，我心里好受了一点，还打算第二天去找她问问。


3


去到珠妹家时，大人都不在，估计到田地里干农活去了，她也不在，我只见到她的亲哥大康，一个同样比我大五岁的男孩。


大康叫我别走，先等等，珠妹很快就回来的。我傻乎乎地什么也没多想，乖乖地坐在客厅里，然后稀里糊涂地被大康带到他们家的二楼。


他脱下我的裤子，让我张开双腿坐在二楼客厅的地板上，我又照做了。他转头扒拉出一箱子晶莹剔透的玻璃仪器，应该是他在化工厂里工作的父母带回来的。


大康试图把长长的玻璃试管塞进我的下体，但他摆弄了好一会儿发现插不进，找不到他想研究的那个洞。


我开始不安起来，但又不知道这意味着什么。双腿大张成M字型，把下体完完全全地暴露在一个男孩面前，没有羞耻感，只是觉得凉飕飕的。


他把玻璃仪器收好后，接着拿出了一盒彩色画笔，“退而求其次”地在我下面作画。画了有大半个小时吧，我的腿张得有点酸了。


via / Dean Stuart


不记得他是怎么放我走的，只记得我马上跑回家，刚好那天妈妈休息，我脱下裤子给她看。


妈妈拿了一个红色的塑料小盆子，倒进烧开的热水，再用井水匀成温水，叫我张开腿坐在里面。


她一边用毛巾仔细轻微地给我擦拭，一边抬头问我是谁画的，眼里满是心疼。我告诉她，是大康弄的。


妈妈又问：“小豹他们两兄弟有没有弄过你这里？”


我摇摇头，不敢说，似乎感觉到眼前这事已经很不好了，再说出来，不知道会发生什么，也害怕小豹他哥会打我。


她没再说什么了，只是继续给我洗，盆子里的水有蓝有绿有黄有红，倒了再换，用了好几盆水才洗干净。末了，她轻声叫我不要再和大康他们玩。

 

妈妈没有去找他的父母。


4


后来，我上小学时搬到镇上住，但这些事还一直烙在心里。


小学二年级的一个晚上，我鼓起勇气告诉妈妈小豹和他哥对我做的事。


她起先以为我在开玩笑，我重复了一次，妈妈变得认真起来。她眼睛把我抓紧，问村子里都有哪些男孩弄过我的下面。


我敏锐地察觉到妈妈态度忽然转变的原因——这可能是件严重的事情。她从来没用这种语气和我说过话。


但我又不敢把事情说出来了，害怕妈妈会去找他们的父母大闹一场讨个说法，到时候别人的指指点点闲言碎语，就会将我和耻辱感捆绑一辈子。


妈妈被我乱七八糟不着调的话糊弄过去后，我再也没和别人提起过这些事。但我开始抗拒回村。


即使非要回去，碰到他们邀请我一起玩游戏时，我都会断然拒绝，担心又是那种暧昧不明的脱裤子游戏。


我逐渐长大，那些事情也慢慢地深藏在心底了。


直到上初中时，伴随着胸部发育月经来潮，我对性的好奇猛地窜了出来。


当时宿舍里有个女孩，她从家里带了一些手掌大的小黄书回来，分给我们看。写的都是以爱情为次、上床为主的故事，虽然描写性爱场面的文字比较隐晦，比较艺术化，但我也大致明白这是什么一回事。


其中“强奸”，“处女”，“贞操”这样的字眼最能勾起我心中压抑已久的秘密。而且，她们在聊到强奸案时，我心里总会咯噔一下，不敢出声。


睡在我上铺的舍友感慨道：“真的想象不出被强奸的女孩会有多痛苦，如果我遭遇了这种事，可能会死了算了。不过，幸好我们全宿舍的人都没经历过。”一阵说不清道不明的难受在我心里掠过，但说不出口，也不敢说出口。

我一直在想，小时候那些事到底算不算强奸，我到底是不是处女？


那时候的我觉得还没结婚就破了处的肯定是不要脸的坏女孩，第一次必须留到新婚之夜。如果我的处女膜真的没有了，那以后怎么和丈夫交代啊？


小黄书里写到第一次都会很痛。但当时小豹的荫茎送进我的下体时，我并没有任何痛感，也没有流血，甚至有点喜欢，因为事后我会想念那种感觉。


趁着家里没人时，隔着裤子将私处抵在塑料椅子座面的圆润一角摩擦，会有种奇异的快感。


我还自行上网搜索，原来儿童时的处女膜很厚，且藏得比较深，一般不会轻易破裂，那么我肯定还是处女啦。


5


自我安慰地高兴了好一段时间，没想到有天放学坐公交车回家，我一回头，久未谋面的小豹他哥居然正站在我的斜右侧方。


他并没有长得很高大，在同龄男生里显得瘦弱苍白，鼻梁上架着一副黑框眼镜，手指细长得有点不像话，浑身上下携着一种暧昧不明的气质。


被摁住脑袋，被言语威胁着去亲他下体时的场景，猛地跳了出来，连同好多次裤子褪下至脚踝的瞬间。


各种关于性的回忆错乱地交集在一起，最后连结成一个定论，我小时候就是被强奸了，我已经不是处女了，我脏了。于是，我噙着眼泪抿紧嘴角无声地哭了一路。


via / 于三门


后来，上高中时，某天去外婆家玩，客厅里聚集了好几个中年女性，我在一旁侧耳听她们聊着家家户户的八卦。


她们说：隔壁村有两个五六岁大的孩子，一男一女，脱了裤子在田埂上模仿大人性交的姿势。


路过的大人看到后告诉两个孩子的父母，结果小男孩被自己的爸爸毒打了一顿，又是扇耳光又是踹脸，还拿藤条把孩子的身体鞭得处处是伤。


小女孩则被家人指着骂：“才这么小就是那种勾男人上床的骚货”。


然后她们一脸感慨地评论道，“估计那两小孩只是看到黄色影片，模仿而已，才这么小，怎么会懂这些事啊？再说他们的家长也是有病，打也不用打得这么凄惨吧。”


我听得心里一阵阵后怕，庆幸当年奶奶并没有撞见我和小豹之间的性游戏。不然她肯定会辱骂我，使我被性羞耻和性恐惧捆绑着长大。


可以说，父母对我的性教育简直为零，我基本所有知识都是自己从书上、网上了解到的。


现在，我是一名心理专业的大二学生，了解到了弗洛伊德的泛性论以及发展心理学，才知道其实五六岁时探索式的儿童性游戏是很正常的，以及儿童会有寻找快感的自慰行为。



比起其他女孩自述的童年被性侵经历，我显得有些特殊，毕竟我当时并没多大痛苦，只是疑惑为什么他们要那样做。


小豹的荫茎塞到我的下体时，其实并没有插进去，而且也插不进去，只是软软的小荫茎，是一种儿童探索性的行为，并不出于性欲或肮脏的强奸念头。


而小豹他哥要我亲他的下体，确实给当时的我带来强烈的屈辱感，是因为他威逼恐吓的态度，同时要我亲的居然是撒尿的地方！我感到怪异但又不得不强迫自己顺从，我俯下身的那一刻还带着些尴尬，为他自己不觉得尴尬而尴尬。


via / Geoff McFetridge


我现在重新挖出那段经历，认真审视大康和他对我的行为，算是完全释怀了，定不定义成性侵已经不重要了，因为我可以正视这些事，并毫无顾忌地告诉男朋友。


但如果当初小豹他哥的年龄再大一点，能够在性欲的作用下勃起；如果大康不管不顾地用玻璃试管猛捅我的下体……那我今天的诉说就是一个彻头彻尾的性侵受害者的痛苦故事了。

 

女孩子的童年啊，真是一场巨大的冒险。
