01.

莓辣的士多店要开始改版升级啦


莓辣士多店是莓辣自己运营的小店，里面有中二且酷、好看且实用，且不乏趣味性的商品。



莓辣现招聘电商运营官（全职）、电商运营（实习） 各1名。


希望你来和我们一起经营有趣的士多店！


 电商运营官（全职）-深圳线下办公 


岗位描述：

1. 主要负责微信有赞商城选品工作，熟悉小程序商城各版块的选品需要，针对各类具体需求选出适合商品；

2. 对接厂家和公司，与不同品牌接洽，完成产品分销，达成合作；

3. 通过数据分析，不断优化选品，优化内容，提升电商转化；

4. 负责电商/推广产品框架和长文的撰写，挖掘产品卖点、洞悉粉丝特性，稿费在工资之外另外结算；

5. 电商大促活动的策划和执行。


岗位要求：

1. 本科及以上学历，专业不限；

2. 有不少于一年的电商运营的经验，有赞运营相关经验优先；

3. 擅长电商文案、推广文案的撰写；

4. 耐心细致，逻辑思维能力强， Excel熟练; 善于沟通，具备较强的责任性和同理心，凡事能够跟进到底。


我们提供：

1. 不算太高但是也还可以的固定薪资

2. 额外的软文稿费

3. 五险一金齐全

4. 不定时掉落的办公室福利

5. 一群志同道合的伙伴


投递要求：

商品推广文案作品集+个人简历发送到邮箱：bigbin@meila.love  
邮件主题：【电商全职】+姓名+手机或微信

投递有效期截止到2021年5月



 电商运营（实习）-深圳线下办公 


岗位描述：

1. 负责部门电商平台所需商品的上架处理，优化商品信息；

2. 处理线上店铺顾客咨询、订单、发货、售后；

3. 代表公司与合作厂家、快递方对接，保证信息准确、流畅；

4. 负责线上电商社群运营；

5. 协助电商运营官完成电商活动策划内容；

6. 协助电商运营官完成电商推荐文案的撰写。



岗位要求：

1. 本科及以上学历在校生（gap year亦可），电子商务、市场营销、心理学等专业优先；

2. 性格开朗外向，优秀的语言表达能力和沟通能力，喜欢与人沟通；

3. 熟悉微信排版，熟练使用office办公软件；

4. 对电商工作有热情；

5. 线下办公时间不少于一周3日。


我们提供：

1. 还不错的实习薪资

2. 莓辣er的内部培训资源（文案、运营等）

3. 不定时掉落的办公室福利


投递要求：

个人简历发送到邮箱：bigbin@meila.love
邮件主题：【电商实习】+姓名+手机或微信

本岗位招募长期有效



02.

「我向许多人打听过你」项目HR志愿者 - 可线上


我们计划在2021年8月前后，举办「我向许多人打听过你」主题活动。我们希望通过这个活动，探索实践性教育的新方式，同时，为大家创造一个有趣的平常谈性的环境。


项目简介可以在点这里看！！


我们希望3月招募一个HR志愿伙伴，作为展览策划及执行过程当中的支持者。


职责包括：

1. 协助进行团队成员的流动管理，包括新成员招募、面试筛选、待定池管理等；

2. 协助对团队成员进行激励、支持，帮助更好地完成展览工作；

3. 优先3月可到岗，项目结束时间为2021年8月。


岗位要求：

1. 人力资源、艺术管理、项目管理等专业优先；

2. 时间、精力较充沛；细心、热情、执行力强；

3. 有主见，对人力资源管理、项目管理、人际沟通有兴趣；

4. 有经验及个人思考的想法更佳！


你将收获：

1. 对于性教育、女性主义等议题有更加深入的了解；

2. 收获一段干货满满的项目实践经验。我们会将过往的项目经验毫无保留的分享，让我们一起打造更棒的项目；

3. 与莓辣以及莓辣亲密的朋友们、机构们一起工作；

4. 相关的证书（志愿者证明、实习证明等）；

5. 亲眼见证莓辣第一场大型展览的从0到1！


投递要求：

个人简历发送到邮箱：bigbin@meila.love
邮件主题：【展览HR志愿者】+姓名+手机或微信

本岗位招募长期有效
